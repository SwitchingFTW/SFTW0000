SFTW

SWITCHING FOR THE WIN ® is an organisation designed to bring professional grade equipment to consumers trough the Sharing Economy.

Products can be designed, manufactured, tested and delivered by different users. The user database is currently managed manually at SFTW Discord server .

This online unrestricted user service will provide Open Source user guidelines, as well as automatic DRC (design rule check) macros for compliance testing. By default, all macros will be written in Python3, since it is compatible with most Engineering Open Source Software, such as FreeCAD, KiCAD, or LibreOffice.
